O que é o CSS e como usar? - parte 1

CSS: Cascating Style Sheets.

Basicamente o CSS é a estilização do seu site, enquanto o HTMML é o conteúdo em sí

existem 3 formas de inserir o CSS na página.

inline ; externo e interno.

majoritáriamente usaremos externo para fins de padronização e organização.

Para os CSS externo funcionar tem que ser linkado no HTML


A base geral do CSS

existem formas de selecionar elementos no CSS.

podemos selecionar com o atributo, ex:
h1{parametros}

ou com o id, ex:
#sahur{parametros}

ou com a classe, ex:
.class{parametros}
